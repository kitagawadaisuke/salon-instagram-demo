# salon-instagram-demo

美容室向け **Instagram運用自動化** の提案・デモページ（静的HTML）。

画像と「今回伝えたいポイント」を用意するだけで、投稿文・ハッシュタグの自動生成 →
指定スケジュールでのInstagram自動投稿 → 結果記録までを自動化する仕組みを再現したデモです。

## ページ構成
- `index.html` … 提案ページ（メイン）
- `sheet.html` … 管理シート（Spreadsheet）のデモ

## 公開
GitHub Pages で公開（このリポジトリの Settings → Pages）。

## 技術
Google Spreadsheet + Google Apps Script による自動化を想定。
ページ自体は依存なしの静的HTML/CSS/JS。
